# 微信小程序代码审核与提审准备

## 1. 提审前检查清单

### 1.1 基础配置
- [x] AppID 已配置（`mini-app/app.json`）
- [x] 服务器域名已配置（微信后台 → 开发管理 → 开发设置 → 服务器域名）
  - request 合法域名：`https://your-api-domain.com`
  - uploadFile 合法域名：`https://your-api-domain.com`
  - downloadFile 合法域名：`https://your-api-domain.com`
- [x] 隐私协议已配置（微信后台 → 设置 → 基本设置 → 服务内容声明）

### 1.2 功能完整性
- [x] 首页作品展示（瀑布流 + 分类筛选）
- [x] 分类浏览页面
- [x] 作品详情页（图片轮播 + 评论 + 点赞）
- [x] 搜索功能
- [x] 个人中心页面
- [x] 微信登录（静默登录）
- [x] 用户信息授权（头像 + 昵称）

### 1.3 内容合规
- [ ] 无违禁词（政治、色情、暴力等）
- [ ] 无诱导分享/关注行为
- [ ] 无虚假/夸大宣传
- [ ] 无侵权内容（图片、文字等）

### 1.4 用户体验
- [ ] 页面加载时间 < 3秒
- [ ] 无空白页面（loading 状态完整）
- [ ] 无明显 bug（闪退、卡顿等）
- [ ] 适配不同屏幕尺寸

### 1.5 隐私合规
- [ ] 首次使用弹出隐私协议
- [ ] 收集用户信息前明确告知用途
- [ ] 提供退出/注销选项
- [ ] 不过度收集用户信息

## 2. 提审流程

### 2.1 微信后台配置
1. 登录微信公众平台（https://mp.weixin.qq.com）
2. 进入「开发管理」→「开发设置」
3. 配置服务器域名（request/uploadFile/downloadFile）
4. 进入「设置」→「基本设置」→「服务内容声明」
5. 填写隐私协议和用户指引

### 2.2 上传代码
1. 使用微信开发者工具打开项目
2. 确认 AppID 正确
3. 点击「上传」按钮
4. 填写版本号（如 `1.0.0`）和项目备注
5. 上传成功后，在微信后台「版本管理」可见

### 2.3 提交审核
1. 在微信后台「版本管理」找到已上传的版本
2. 点击「提交审核」
3. 填写审核信息：
   - 功能页面：选择首页
   - 测试账号：提供管理员账号（如有登录功能）
   - 补充说明：简要描述小程序功能
4. 确认提交

### 2.4 审核注意事项
- 审核时间：通常 1-7 个工作日
- 常见驳回原因：
  - 功能不完整（页面空白、无法操作）
  - 内容违规（敏感词、侵权）
  - 诱导行为（强制分享、关注）
  - 隐私问题（未告知收集信息用途）
- 驳回后：根据驳回原因修改，重新提交

## 3. 发布流程

### 3.1 审核通过后
1. 在微信后台「版本管理」点击「发布」
2. 选择发布范围（全量/指定用户）
3. 确认发布

### 3.2 发布后检查
- [ ] 线上版本功能正常
- [ ] 无严重 bug
- [ ] 性能表现良好

## 4. 后续维护

### 4.1 版本更新
1. 修改代码并本地测试
2. 上传新版本（递增版本号）
3. 提交审核
4. 审核通过后发布

### 4.2 监控告警
- 监控接口响应时间
- 监控错误率
- 监控用户反馈
- 定期清理过期数据

## 5. 常见问题

### Q1: 服务器域名配置后仍提示域名不在合法域名列表
A: 检查是否配置了正确的域名前缀（https://），检查是否有拼写错误，检查域名是否已备案。

### Q2: 上传图片失败
A: 检查 uploadFile 合法域名是否配置，检查图片大小是否超过限制（10MB），检查网络是否正常。

### Q3: 登录接口调用失败
A: 检查 AppID 和 AppSecret 是否正确，检查服务器是否正常运行，检查网络是否正常。

### Q4: 审核被驳回怎么办
A: 查看驳回原因，根据原因修改代码，重新提交审核。如对驳回有疑问，可联系微信客服。
