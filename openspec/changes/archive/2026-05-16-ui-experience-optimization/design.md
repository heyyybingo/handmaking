# UI体验优化设计文档

## 设计依据

基于Stitch设计稿（项目ID: 928606282179741372）的6个屏幕设计。

## 1. 作品详情页优化

### 1.1 图标位置精校版
**文件**: 01-detail-icons.png

**改动点**:
- 顶部图片区域：全宽展示，高度400rpx
- 作品信息区：标题、分类标签、作者信息
- 描述区：支持富文本，段落间距优化
- 底部操作栏：图标位置居中，间距均匀

### 1.2 沉浸式转换版
**文件**: 05-detail-immersive.png

**改动点**:
- 图片轮播支持沉浸式模式（全屏+状态栏适配）
- 向上滑动时图片渐隐，信息区渐显
- 底部操作栏固定，半透明背景

### 1.3 富文本排版版
**文件**: 06-detail-richtext.png

**改动点**:
- 支持富文本内容展示
- 段落间距：24rpx
- 图片圆角：16rpx
- 引用块样式：左侧4rpx铜色边线

## 2. 个人中心页面

**文件**: 02-profile.png

**布局结构**:
```
┌─────────────────────────┐
│      温暖匠心（标题）      │
├─────────────────────────┤
│      [用户头像]          │
│      用户昵称            │
│      点击完善个人信息 >   │
├─────────────────────────┤
│  128  │  45  │  12     │
│ 我的点赞│我的关注│我的留言  │
├─────────────────────────┤
│  ♡ 收藏的作品      >    │
│  ⚙ 设置            >    │

├─────────────────────────┤
│  🏠  📂  👤            │
│  首页  分类  我的        │
```

**样式规范**:
- 背景色：#F5F5F5
- 卡片背景：#FFFFFF
- 圆角：24rpx
- 阴影：0 4rpx 20rpx rgba(0,0,0,0.06)

## 3. 首页瀑布流动效

**文件**: 03-home.png

**优化点**:
- 卡片入场动画：从下往上淡入，间隔50ms
- 图片加载：骨架屏 → 渐显
- 点击反馈：卡片缩放0.98 → 恢复

## 4. 意向面板动效

**文件**: 04-intent-panel.png

**优化点**:
- 弹出动画：从底部滑入，时长300ms，ease-out
- 遮罩层：渐显，rgba(0,0,0,0.5)
- 意向类型选择：点击时铜色边框渐显
- 提交按钮：loading状态旋转动画

## 5. 色彩系统（沿用现有）

- 主色：#C4956A（暖铜色）
- 辅助色：#8B6F4E（深木色）
- 背景色：#F5F5F5
- 卡片背景：#FFFFFF
- 分割线：#E5E5E5

## 6. 字体系统（沿用现有）

- 标题：28rpx, 600
- 正文：28rpx, 400
- 辅助：24rpx, 400
- 标签：20rpx, 500
